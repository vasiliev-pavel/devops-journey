# Homelab · Linux & DevOps Engineering

Практические инженерные лаборатории на собственном homelab-стенде: Linux-серверы, сети,
контейнеризация, CI/CD, инфраструктура как код и мониторинг. Каждая лаба — рабочий
результат с конфигами, скриптами и разбором решений.

**Стенд:** VirtualBox · Ubuntu Server 24.04 LTS (headless) · управление по SSH
**Подход:** infrastructure-as-code, версионирование конфигов, документирование решений

## Технологии

**ОС и администрирование:** Linux (Ubuntu Server), systemd, управление пользователями и правами
**Сеть:** SSH, TCP/IP, ufw/iptables, nginx
**Автоматизация:** Bash, cron, Ansible
**Контейнеры и оркестрация:** Docker, docker-compose, Kubernetes
**CI/CD:** Git, GitHub Actions
**Observability:** Prometheus, Grafana
